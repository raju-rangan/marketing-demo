# Shorts & Social Media Overrides

If the user has requested short-form vertical content (Shorts, Reels, TikToks), you must follow the core `slidecast-production` workflow, but apply these mandatory overrides:

## 1. Technical Constraints
- **Duration**: When calling `generate_slidecast_storyboard`, you MUST set `duration_seconds=60` (or less).
- **Aspect Ratio**: You MUST set `aspect_ratio="9:16"` to generate vertical video suitable for mobile devices.

## 2. Anatomy of a High-Converting Short
A successful social media short is vastly different from a corporate slidecast. You must actively follow these best practices:

- **The 3-Second Hook (CRITICAL)**: The very first scene must visually and audibly grab attention instantly. No slow build-ups, no generic title cards, and no "Welcome to..." intros. Start immediately with a bold claim, a surprising statistic, or a direct question.
- **Visual Pacing & Readability**: Vertical feeds demand constant visual stimulation. Keep text overlays massive, punchy, and minimal (1-5 words max). Absolutely NO dense paragraphs, complex charts, or textbook-style bullet points.
- **Composition & Framing**: The primary subject and core text must be centered. Avoid putting crucial visual information on the far right (where Like/Comment buttons live) or the bottom fifth of the screen (where captions and usernames live).
- **Narrative Structure**: Tell a micro-story. Use structures like:
  - *Problem -> Agitation -> Solution*
  - *"Did you know?" -> Explanation -> Result*
  - *Myth -> Reality -> Proof*
- **Visual Dynamics**: Prioritize bold, eye-catching visual scenes (expressive people, dramatic lighting, bold typography, implied motion) rather than sterile infographics.
- **The Call to Action (CTA)**: The final scene must end sharply with a clear directive (e.g., "Link in bio", "Read the full report", "Subscribe for more insights"). Do not let the video fade out slowly.
